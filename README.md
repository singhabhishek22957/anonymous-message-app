
## Getting Started

First, run the development server:

```bash
npm run dev
or
npm run start
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

This application allows users to receive anonymous reviews and feedback through a unique shareable link.

## How it works

Each user gets a personal unique link
Anyone can visit the link and send feedback anonymously
No login is required for sending messages
All messages are visible only to the link owner in their dashboard

## 🔐 Authentication & Security

Built with Next.js App Router

Authentication handled using NextAuth (Credentials Provider)

Uses JWT-based session strategy

Passwords are securely hashed using bcrypt

Protected routes for authenticated users only

# 🤖 AI-Powered Suggestions (OpenAI)

To improve user experience, the platform integrates OpenAI:

💡 Suggests smart and meaningful feedback

✍️ Helps users write better anonymous messages

⚡ Enhances quality of communication

🧠 Useful when users are unsure what to write

## 🚀 Features

🕵️ Fully anonymous messaging system

🔗 Unique user profile link

📥 Private dashboard to read messages

🔐 Secure login & authentication (NextAuth + JWT)

🤖 AI-based message suggestions (OpenAI)

📱 Responsive and modern UI (Next.js + Tailwind)
